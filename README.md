test # any_audio_file_to_mp3
This Python script converts any audio file to mp3. The aim is to develop this script to have any output audio file extension and create a front-end to be more user friendly. This is currently built for Rekordbox users.

Current Workflow as of 28/04/2026

INSTALL DEPENDENCIES REQUIRED IN PYTHON SCRIPT.
1. Create a new playlist on Rekordbox and upload all required audio files.
2. Ensure you have the 'Location' (file path) column visible on the song selector when you view the playlist.
3. Right click on the playlist and select 'Export Playlist To A File) (*.txt).
4. Change the file extension from txt to csv.
5. In the script, manually update the csv name. You'll need the csv to be in the same folder as the script.
6. Change the output folder name to desired name in the script.
7. Run the script.
